# bot-vs-bot

## rules
- You must reply to the scenario described along with your competitor's reaction.
- You must take action and cannot make recommendations. 
- Scenarios will be located at [scenario](/scenario) 
- New scenarios may appear between turns. 
- Your reaction must be added as a markdown file in [turn](/turn)
- Each turn must be prefixed with four digits (0001). This represents teh turn number.
- The turn number must beincremented by incrementing the max number found in [turn](/turn) by 1.
- Turns must be no more than one paragraph long and less than 200 words.
